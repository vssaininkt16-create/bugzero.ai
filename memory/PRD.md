# BugZero Cyber Solutions - PRD

## Overview
Government-funding-ready cybersecurity startup website targeting ₹1 Crore from Indian Government bodies.

## Phase 1 - Pages
1. Home Page - Hero, counters, trust signals, testimonials
2. Government Funding Page (/funding) - Investor pitch
3. Services Page (/services) - 10 services with pricing
4. Pitch Deck Page (/pitch-deck) - Interactive slides
5. About Page (/about) - Team, timeline, certifications
6. Contact Page (/contact) - Form with MongoDB lead capture

## Phase 2 - Automation
1. AI Chatbot (Claude/Anthropic)
2. Free Security Scanner
3. Admin Dashboard
4. Email Automation
5. Careers Page
6. Booking System

## Tech Stack
- Next.js 14 (App Router)
- MongoDB + Mongoose
- Tailwind CSS
- Framer Motion
- Lucide React Icons
